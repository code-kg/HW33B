1. Error - это не исправимая ошибка которая явялется наследником от Throwable
RuntimeException и его подклассы являются unchecked исключениями
Checked Exception нужно обработать самим в ручную
2. Ключевое слово throw в Java используется для явного выброса исключения из метода или любого блока кода.
Мы можем бросить либо проверенное, либо непроверенное исключение. Ключевое слово throw в основном используется для создания пользовательских исключений.
3. Unchecked не обрабатывемые исключения
4. Блок catch выполняется только в случае возникновения исключения в блоке try
Блок finally выполняется всегда после блока try (-catch), если выброшено исключение или нет.
5. если исключения имеют отношения родитель-потомок, 
блоки catch должны быть отсортированы сначала по наиболее конкретным исключениям, 
а затем по наиболее общим.
